## Milestone 2. 솔저형 특수 스킬
### 목표
- 질주/로켓/생체장 3종 스킬을 전투 루프에 통합

### 체크리스트
- [ ] 질주 버튼 연동, 속도 1.5배 반영
- [ ] 사격 시 질주 즉시 해제 로직 구현
- [ ] 로켓 Sphere 발사 프리팹/물리 이동 구현
- [ ] 충돌 폭발 + `OverlapSphere` 범위 대미지 구현
- [ ] 생체장 생성, 5초 지속, 초당 20 회복 구현
- [ ] 스킬 동시 사용/재사용 시 예외 동작 점검
