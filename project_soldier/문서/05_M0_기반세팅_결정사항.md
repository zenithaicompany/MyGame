# Milestone 0 기반세팅 결정사항

## 1) 엔진/렌더링 버전 확정
- Unity: `2022.3 LTS` 권장
- Render Pipeline: `URP`
- 템플릿: `3D (URP)` 또는 기존 프로젝트에 URP 패키지 적용

## 2) 입력 시스템 결정
- 입력 시스템: `New Input System` 채택
- 선택 이유
  - 모바일 터치/멀티터치 처리에 유리
  - UI 이벤트 시스템과 연동이 명확
  - 추후 키보드/패드 폴백 확장 용이

## 3) 프로젝트 기준 디렉터리
- `Assets/Scenes`
- `Assets/Scripts`
- `Assets/Prefabs`
- `Assets/UI`
- `Assets/Art`
- `Assets/Art/Characters`
- `Assets/Art/Environment`
- `Assets/Art/UI`
- `Assets/Art/VFX`

## 4) 품질/플랫폼 초기 기준
- 타겟 프레임: 권장 60fps, 최소 30fps
- 우선 플랫폼: Android 기준 검증 후 iOS 확장
- 품질 프리셋: 모바일 미들급 기준으로 시작(그림자/후처리 최소)

## 5) 다음 단계 진입 조건
- 테스트 씬 1개 생성 완료
- 플레이어 시작 위치, 바닥, 타겟 박스 배치
- 조이스틱/시점/사격 입력 연결 준비 상태
