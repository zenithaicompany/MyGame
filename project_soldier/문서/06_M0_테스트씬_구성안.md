# Milestone 0 테스트 씬 구성안

## 씬 이름
- `Assets/Scenes/Testbox_MVP.unity`

## 씬 목적
- Milestone 1 구현 전, 이동/조준/사격/스킬 기능을 빠르게 검증하기 위한 베이스 씬

## 기본 계층 구조(Hierarchy) 권장
- `Directional Light`
- `Global Volume`(선택, 최소 세팅)
- `Ground` (Plane 또는 Cube)
- `Environment_Root`
  - `Wall_01 ~ Wall_04` (맵 외곽)
  - `CoverBox_01 ~ CoverBox_10` (엄폐 박스)
- `Target_Root`
  - `TargetBox_01 ~ TargetBox_08` (`Tag: Target`)
- `PlayerSpawn`
- `Player` (FPS 카메라 포함)
- `UI_Root` (Canvas/EventSystem)

## 오브젝트 배치 기준
- Ground: `30 x 30` 이상
- 외곽 벽: 플레이어 이탈 방지
- TargetBox: 5m/10m/15m 거리별로 분산 배치
- PlayerSpawn: 중앙에서 약간 후방 배치

## 컴포넌트 체크
- Target 박스
  - Collider 존재
  - `Target` 태그 설정
- 로켓 충돌 검사용 벽
  - Collider 존재
  - Static 처리(권장)
